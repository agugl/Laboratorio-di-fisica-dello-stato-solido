# Codici Python - Risposta Spettrale

Questa cartella contiene gli script Python per l'elaborazione dei dati.

Eseguire i codici in quest'ordine:

1. `01_J.py` - Calcolo densità di corrente J(λ)
2. `02_Pua.py` - Caricamento potenza per unità di area P_ua(λ)
3. `03_rapporto.py` - Calcolo rapporto J/P_ua
4. `04_EQE.py` - Efficienza Quantica Esterna EQE(%)
5. `05_Tauc.py` - Tauc plot per gap indiretto

Tutti i codici sono progettati per essere eseguiti su Google Colab.
